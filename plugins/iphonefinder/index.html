Le plugin iphonefinder permet de géolocaliser un iPhone en utilisant le service Find My iPhone d'Apple

Les phrases magiques sont définies dans le fichier de grammaire XML iphonefinder.xml et sont à modifier selon votre convenance (en changeant le prénom pour commencer, sauf si vous vous appelez Franck! :)

  • Sarah, où est Franck?
  • Sarah, où es mon iPhone ?
<item>Franck<tag>out.action.who = "Franck"</tag></item>

Par défaut, le plugin localise le premier iDevice lié au compte. Si il y en a plusieurs, il suffit d'indiquer le numéro du device dans le fichier xml :

<item>Franck<tag>out.action.who = "Franck"</tag></item>

Il suffit de renseigner vos identifiants iCloud dans la configuration du plugin, et ca marche!

Il est possible d'affiner le résultat en indiquant le niveau de précision dans le tag "what" :
- "Distance" pour la distance,
- "City" pour la ville,
- "Address" pour l'adresse complète
- "Mix" pour la ville et la distance
- "Alarm" pour faire sonner le device
- "Battery" pour connaitre le niveau de batterie d'un device ***NEW*** (Par Alban)
- "Showme" pour afficher le device sur une carte en ChromeLess (*** EXPERIMENTAL ***) (Par Alban) 

<item>ou est<tag>out.action.what = "Address"</tag></item>

Si vous voulez que SARAH vous donne la distance qui sépare l'utilisateur à localiser d'un endroit (la maison) il faut saisir les coordonnées GPS de l'endroit
dans la configuration du plugin (latitude_sarah et longitude_sarah). Pour les trouver, il suffit d'aller sur Google Maps,
de localiser l'endroit, et de faire un clic droit sur le marqueur rouge et "Plus d'infos sur cet endroit".
les coordonnées (latitude,longitude) sont indiquées dans la barre d'adresse de Maps.


Mode Guessit: (Par Alban)

le Tag "mode" peut contenir "guessit" pour que SARAH tente d'identifier l'utilisateur (reco facial ou pitch audio) quand il dit "mon téléphone". Si elle ne trouve pas de profil, elle prend le 1er device du 1er compte iCloud.

Surveillance de batterie: (*** EXPERIMENTAL ***) Si vous paramétrez le paramètre monitorbatterylvl à 1, SARAH vérifie le niveau de batterie (toutes les heures) et vous averti si elle descend en dessous de 10% La fréquence de vérification et le niveau minimal peuvent être modifiés dans iphonefinder.js NOTE IMPORTANTE : La récupération des infos se faisant par un chemin non officiel (via les URL d'Apple), il n'est pas prévue par Apple que l'on demande le niveau de batterie à intervalle réguliers. Ceci est donc à utiliser pour tests uniquements. De plus, il se peut que des demandes répétées aient un impact sur la batterie de l'iPhone. :)
